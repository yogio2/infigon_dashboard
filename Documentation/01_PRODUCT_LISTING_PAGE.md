# Task 1: Product Listing Page

## Route
`/` (Home page)

## Objectives
Create a responsive product listing page that fetches and displays products from an external API with proper loading and error handling.

## Requirements

### Data Fetching
- **API Endpoint**: `https://fakestoreapi.com/products`
- Fetch products on page load
- Handle API calls efficiently

### Display Information
Display each product in a responsive grid with the following information:
- Product image
- Title
- Price
- Category

### Loading State
- Implement a loading indicator while fetching products
- Skeleton cards recommended for better UX

### Error Handling
- Implement error state handling
- Display user-friendly error messages
- No silent failures

### Grid Layout
- Products should be displayed in a responsive grid
- Adapts to different screen sizes (mobile, tablet, desktop)

## Acceptance Criteria
- [ ] Products are fetched from the API successfully
- [ ] All required product information is displayed
- [ ] Loading state is visible while fetching
- [ ] Error state is handled gracefully
- [ ] Grid is responsive across all screen sizes
